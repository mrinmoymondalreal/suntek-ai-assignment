# React + TypeScript + Vite + Tailwind + React Router + ShadcnUI

## To get started with the Template

### Clone the Repo

```bash
degit https://github.com/mrinmoymondalreal/Custom-React-Template
```

### Install dependencies

```bash
pnpm install
```

or

```bash
npm install
```

### Start the app

```bash
pnpm run dev
```

or

```bash
npm run dev
```
